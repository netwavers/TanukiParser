# TanukiParser 🐾

**The Next-Gen Lyrics Parsing Engine powered by Rust & EBNF.**

[![GitHub Pages](https://img.shields.io/badge/Live-Demo-blue?style=for-the-badge&logo=github)](https://netwavers.github.io/TanukiParser/)

TanukiParserは、EBNFベースの強力な文法定義とRustの高速性を組み合わせた歌詞解析エンジンです。

## ✨ 特徴

- **EBNFベース**: 複雑な歌詞構造を直感的に定義可能。
- **超高速**: Rust Coreによるミリ秒単位のパース処理。
- **マルチ出力**: PythonやC#へのコード生成に対応。
- **ドキュメント充実**: チュートリアルやリファレンスを完備。
